# Blindnet sdk test

A projet to test Vite (https://vitejs.dev) & Blindnet.io javascript SDK (https://blindnet.io)

## Requirements

I prefer to use `pnpm` instead of `npm` or `yarn` to keep space on hard disk.

## Installation

`pnpm install`

## Development usage

`pnpm run dev`

## Production mode

`pnpm run build`
